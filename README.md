# 🎨 Personal Art Showcase Website

Welcome to my personal art showcase website!
This project is a creative space where I display my artwork, experiment with design, and express my artistic style.

## 🌟 Features

* 🖼️ Gallery to showcase my artwork
* 🎞️ Smooth animations and transitions
* 📱 Responsive design (works on mobile & desktop)
* 🎨 Aesthetic and minimal UI
* 💬 Interactive elements using JavaScript / jQuery

## 🛠️ Tech Stack

* HTML5 – Structure
* CSS3 – Styling & animations
* jQuery – Interactivity

## 📂 Project Structure

```
project-folder/
│── jquery_project.html
│── shop.html
│── contact.html
│── img/
```

## 🚀 How to Run

1. Download or clone the repository:

```
git clone https://github.com/bobbybyte28/Personal-Art-Showcase-Website.git
```

2. Open the project folder

3. Run `jquery_project.html` in your browser

This website is built to:

* Showcase my artwork
